After unzipping the repository will contain Python scripts, xml, text, cvs, png and svg files. The xml files provide signaling pathways model specification files, that are used to simulate in vitro experiments using NeuroRD. Each model that can be run by NeuroRD requires a model master file, reaction file, initial conditions file, stimulation file and output specification file. Names of model master specification files start with "model". These files can be found in directories starting with "Ca_wave" and "model". Initial conditions filenames start with "IC" and are located in baloon_ER and stacked_ER directories. These directories also contain model files for testing if the model is in steady state. Stimulation files are placed in 1_EPSP directory and 1_EPSP_aging directory. Reaction filenames start with "Rxn", output specification filenames start with "IO" and morphology filenames start with "morph" and will be located in the main directory.

Files containing simulation results have extentions "h5"
